# MSC-Project-Wu


There are 5 folders in this project directory, the first 3 are 3 projects, the fourth Monitoring webpage is the monitoring center page module, directly put this folder into the Tomcat server and the war package of the monitoring center in a directory under.

The last folder contains some necessary jar packages used by java. It is mainly used when the edge node program uses a file named sigar.jar to monitor the running status of the machine, and then you also need to add the files in the sigar-bin folder The corresponding so file in the lib is put into the jdk library of the running server, so that the monitoring program can be used normally.
